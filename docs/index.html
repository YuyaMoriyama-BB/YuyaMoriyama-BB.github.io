WCI 設定事例集

本マニュアルについて

本マニュアルではWCIルータとローカル機器の接続構成やWCIネットワークにコンテンツ・サービスを公開するための設定事例を紹介しています。
別紙 WCI-Portal 利用手順書 と併せて、WCIネットワークへの接続設定にお役立て下さい。
