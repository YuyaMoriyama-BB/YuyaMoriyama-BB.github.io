WCIルータについて
===================

1. 導入機種
------------

以下の機種を提供しております。 
本マニュアル内表記の **WAN** ポート及び **LAN** ポートは導入機種毎に **ポートの対応表** を元に読み替えて下さい。
また、ポートの規格に応じたツイストペアケーブルでの接続を推奨しております。

----

NEC UNIVERGE IX2235
^^^^^^^^^^^^^^^^^^^^
.. image:: ./images/ix2235.jpg
   :scale: 70%

**<ポートの対応表>** 

.. csv-table::

   "マニュアルの表記","対応するWCIルータの物理ポート","ポートの規格"
   "WAN","GE0","1000BASE-T"
   "LAN","GE2(SW-HUB)","1000BASE-T"

※GE2(SW-HUB)は全てのポートが使用できます。


NEC UNIVERGE IX3315
^^^^^^^^^^^^^^^^^^^^
.. image:: ./images/ix3315.jpg
   :scale: 30%

**<ポートの対応表>** 

.. csv-table::

   "マニュアルの表記","対応するWCIルータの物理ポート","ポートの規格"
   "WAN","GE2","10GBASE-T(RJ-45)"
   "LAN","GE3","10GBASE-T(RJ-45)"

----


2. 物理接続
-------------

光回線事業者設置のONU（回線終端装置）とWCIルータをケーブルで接続します。
WCIルータ電源投入後、自動でWCIネットワークに接続されます。

**接続構成図** 

.. image:: ./images/init.png
   :scale: 50%

----

3. 接続確認
-------------

WCIルータがWCIネットワークに正常に接続されているかどうかは機器前面のLEDランプで確認できます。
**PWR** ランプと **VPN** ランプが緑点灯している場合、正常に通信が出来ています。

**IXシリーズ前面LEDランプ**

   :scale: 40%
